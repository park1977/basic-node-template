Files with an .mjs extension.

## env file 

.env sample: 

``` env
# Environment Name
NODE_ENV=development

# Log file rotate
LOG_ROTATE=True

# Server listen to this port
PORT=3000

# Cors
CORS_URL=*

# Databse
DB_NAME=db
DB_MIN_POOL_SIZE=2
DB_MAX_POOL_SIZE=5
DB_HOST=mongo
DB_PORT=27017
```
